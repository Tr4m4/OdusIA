# Luxury Travel Curator v. 3.0

Standalone agent specialized in high-end travel consulting, identifying immersive wellness and luxury experiences.

## Setup
Non è necessaria alcuna installazione di API Key esterne. Questo agente è integrato direttamente in **Antigravity**.

## Utilizzo
Per attivare la curatela, è sufficiente richiederla nella chat con Antigravity facendo riferimento a queste istruzioni. 
L'agente ti richiederà i 4 parametri obbligatori:
- **LUOGO**: La destinazione desiderata.
- **DETTAGLIO SPA ESSENZIALE**: La funzionalità di benessere specifica richiesta.
- **FASCIA PREZZO**: Classe di lusso o budget.
- **FOCUS DEL VIAGGIO**: Tema emozionale o trasversale (Natura, Storia, Design, etc.).

## Struttura della Risposta
L'agente genererà un **REPORT DI CONSULENZA** strutturato con:
- Introduzione autorevole.
- Tabella comparativa delle strutture selezionate.
- Analisi comparativa (Riflessioni del Curatore).
- Call to Action per servizi aggiuntivi.
